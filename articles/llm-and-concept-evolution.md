現状の大規模言語モデル（LLM）による知識処理について考えていたとき、ひとつの仮説に思い至りました。

それは、LLM の知的進化の限界の根源のひとつは  

**「新しい概念を生成し、それを社会的に固定する能力を持たないこと」ではないか**

という仮説。

LLM は新しい表現を作り出すことができます。場合によっては、人間がこれまで明確に言語化してこなかった概念的なパターンを言葉として表現することさえあります。

しかしながら、新しい概念が社会の中で共有され、より深い思考を助ける道具として定着していくプロセスと、現状の LLM との間には、かなりの距離があります。

このギャップは、知性の進化を考える上で、無視できないものなのかもしれない、と考え始めました。


## 概念の形成サイクルと知性の進化

人間の知性の発展を振り返ると、そこにはある種のサイクルが存在しているように思えます。

- 新しい抽象的パターンが発見される  
- それに名前（言葉）が与えられる  
- 社会の中で共有される  
- 人々がその概念を使って思考するようになる 
- その概念を使って思考が深まる
- さらに新しい概念が生まれる  

このサイクルは、科学・哲学・社会制度など、人類の知的発展の多くの側面で見ることができます。

ここでいう「概念」とは、**複雑な現象をひとつの単位として議論できるようにする共有された認知的圧縮** だと言えます。

このような圧縮がうまく機能すると、その概念は思考のインフラの一部になります。

よく知られた例として、**「Meme（ミーム）」** という概念があります。

この言葉は、進化生物学者 Richard Dawkins が著書『利己的遺伝子（The Selfish Gene）』の中で提唱したものです。  
今日では「ネット上に広がるジョーク画像」などを指す言葉として使われることも多いですが、もともとは文化の進化を考えるための概念として導入されました。

Dawkins は、アイデアや行動様式、文化的パターンを **「自己複製する情報単位」** として捉えることを提案しました。  
それらは、生物学における遺伝子のように、人から人へと伝播していくと考えられます。

「ミーム」という言葉が生まれたことで、文化の伝播についての議論は一気に行いやすくなりました。

ミームの拡散、ミームの変異、ミーム間の競争といった形で、複雑な現象をひとつの概念単位として扱えるようになったのです。

このように、新しい概念に新しい言葉を与えることは、**新しい思考の枠組みを生み出すこと** だとも言えます。

適切なラベルが与えられることで、複雑なパターンは

圧縮され  
共有され  
再利用できるようになります。

また、新しいラベルは既存の概念から適度な距離を作り出し、思考を新しい概念空間へと導く役割も果たします。

このように見ていくと、人類の知的進化の多くは次のようなサイクルとして理解できるかもしれません。

**概念の発明 ➡︎ 命名 ➡︎ 社会的共有 ➡︎ 思考の拡張**

人間の知性は、個々人の能力だけで発展してきたのではなく、  
**概念を社会の中で安定させ、累積していくプロセス**によって発展してきたとも言えます。


## LLM はどうか？

現代の大規模言語モデル（LLM）は、非常に高度な言語能力を持っています。

膨大なテキストからパターンを学習し、流暢な文章を生成することができます。  
場合によっては、人間がまだ明確に言語化していない概念的なパターンを言葉として表現することもあります。

この意味では、LLM は **概念の候補を生成すること自体はできる** と言えます。

しかし、現在の LLM の内部構造は、人間社会における概念のあり方とは大きく異なっています。

多くの LLM では

- 語彙は基本的に固定されている  
- 意味表現は内部状態（hidden state や attention pattern）に分散している  
- 通常の利用の中で概念体系が継続的に更新されるわけではない  

といった特徴があります。

つまり、LLM は概念の候補を生成することはできても、  
**概念が社会の中で安定した思考ツールになっていくプロセス** とは距離があります。


## 生成はできるが、固定はできない

人間社会における概念形成は、通常次のようなプロセスを辿ります。

1. 誰かが新しい概念を提案する（しばしば新しい言葉とともに）  
2. それについて議論や批判が行われる  
3. 意味が徐々に収束していく  
4. 概念として安定する  
5. その上でさらに新しい議論が生まれる  

これは、**社会的な概念の交渉と安定化のプロセス** だと言えます。

しかし現在の LLM は、このようなプロセスとは基本的に無関係です。

この観点から見ると、現在の状況は次のように整理できるかもしれません。

LLM は **新しい概念表現を生成することはできる。**

しかし、その概念を **社会的に固定することはできない。**

もし知性の進化にとって重要なポイントが、  

- 共有された概念的ツールを生み出し、
- それを共同で再利用できるようにして、
- 思考の深化を促すこと。

もしそうだとすれば、この違いは、意外に大きいかもしれません。


## 個体知性と集合知性

現在の AI 研究では、モデルサイズや計算量を増やすことで能力が向上する「スケーリング仮説」が広く知られています。
実際、大規模言語モデルの性能は、パラメータ数や学習データ量の増加とともに大きく向上してきました。

つまり現在の AI システムでは、知性のほとんどは **単一のモデルの内部** に存在しています。

この視点から見ると、現代の LLM はまだ、**強力な個体知性の段階** にあると言えます。

しかし、人間の知的進化を振り返ると、知性の発展は単一個体の能力の拡大だけによって起きたわけではありません。

むしろ、人間社会の中で概念が発明され、しばしば専用のラベルが与えられ、共有され、累積していく **社会的プロセス** が重要な役割を果たしてきたと言えます。


## 今後の方向性

もしこの仮説にある程度の妥当性があるなら、AI のさらなる進化には、単にモデルサイズを拡大するだけではない仕組みが必要になる可能性があります。

すでに関連する研究として、いくつかの方向性が存在します。

- Dynamic vocabulary（語彙を動的に拡張する研究）  
- Concept bottleneck models（概念レベルを明示的に扱うモデル）  
- Emergent communication（AI 同士の相互作用による言語の発生）  
- Neural-symbolic AI（概念をより明示的な構造として扱うアプローチ）  

また、AI 同士が、人間を含めて、相互作用しながら概念を共創していくような **社会的システム** が重要になる可能性もあります。

言い換えれば、さらなる AI の発展には **巨大な単一モデル** だけではなく、  
**相互作用するモデルの社会** のようなものが必要になるのかもしれません。


## おわりに

もし人の知性が  
**社会の中で概念を発明し、それを固定していくプロセス**  
によって発展してきたのだとすれば、

AI の未来の知性もまた、  
単独のモデルではなく

**概念が共有され進化していく社会的システム**

その中で発展していくのかもしれない、とか思いはじめた今日この頃です。
